🤖 Comment fonctionne le bot

Le bot utilise une combinaison de 8 stratégies éprouvées (RSI + Bollinger, MACD + RSI, Croisement de moyennes mobiles, Breakout, Mean Reversion, Momentum, Ichimoku, Parabolic SAR) appliquées à 3 niveaux de risque (Prudent, Neutre, Agressif).

Il analyse en temps réel les données de marché provenant de plusieurs sources (Finnhub, Twelve Data, Alpha Vantage) pour générer des signaux d'achat et de vente, adaptés à 4 marchés : Actions, Crypto, Forex, ETF.

🌐 Les API qui alimentent le bot

Notre bot s'appuie sur des API de données financières de qualité professionnelle, qui lui permettent d'avoir une vision complète et à jour des marchés :

  • Finnhub : Données en temps réel (actions, forex, crypto).
  • Twelve Data : Données historiques et de marché (actions, indices).
  • Alpha Vantage : Données fondamentales et historiques (backup).
  • Fincept : Backup ultime pour garantir la continuité du service.

Cette architecture en cascade garantit une disponibilité quasi-totale et une fiabilité maximale.
